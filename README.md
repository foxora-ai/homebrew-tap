# Foxora Homebrew Tap

Install the [Foxora](https://foxora.ai) CLI — the agent terminal — on macOS or Linux:

```sh
brew install foxora-ai/tap/foxora
```

Then run:

```sh
foxora            # open the interactive UI
foxora --help     # all commands
```

The binaries are published from the [foxora-ai/releases](https://github.com/foxora-ai/releases)
GitHub Releases (macOS builds are Developer ID signed + Apple-notarized).

## Updating

```sh
brew update
brew upgrade foxora
```
